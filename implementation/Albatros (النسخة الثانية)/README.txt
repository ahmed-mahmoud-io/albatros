============================================
  ALBATROS Real Estate - طريقة التشغيل
============================================

------------------------------------------------
[1] تأكد من وجود البرامج المطلوبة
------------------------------------------------

  1- .NET 8 SDK
     - حمّله من: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
     - بعد التثبيت، افتح PowerShell واكتب: dotnet --version
     - المفروض يظهر رقم يبدأ بـ 8.x

  2- Visual Studio 2022 (أو أحدث)
     - المهم يكون فيه "ASP.NET and web development" workload
     - LocalDB بتجي automatically مع Visual Studio
     - مش محتاج تثبيت SQL Server منفصل


------------------------------------------------
[2] قاعدة البيانات (LocalDB)
------------------------------------------------

  خلاص — المشروع بيستخدم LocalDB اللي مدمج مع Visual Studio.
  مفيش حاجة تعملها، شغّل بس (الخطوة [3]).

  ملحوظة: لو عايز تستخدم SQL Server Express بدل LocalDB،
  افتح appsettings.json وغيّر:

     Server=(localdb)\mssqllocaldb

  إلى:

     Server=.\SQLEXPRESS

----------------------------------------------------------------


------------------------------------------------
[3] شغّل المشروع
------------------------------------------------

  1- افتح PowerShell أو Command Prompt
  2- غيّر المسار إلى المجلد ده:
       cd "C:\مسار ملفات المشروع"
  3- شغّل الأمر:
       dotnet run
  4- أول تشغيل هياخد دقيقة أو اتنين عشان:
       - يرجع الحزم (NuGet packages)
       - يعمل Build
       - يشغل Migrations (يخلق الداتابيز أتوماتيك)
       - يحط بيانات البذور (Seed data)


------------------------------------------------
[4] افتح الموقع
------------------------------------------------

   افتح المتصفح على الرابط:
     http://localhost:5279

   هتشوف الصفحة الرئيسية. أول مرة هتظهر شاشة ترحيب (Splash screen).
   اضغط أي حته عشان تتخطاها.


------------------------------------------------
[5] تسجيل الدخول بحسابات جاهزة
------------------------------------------------

   الحسابات دي بتتخلق أتوماتيك أول تشغيل:

     Admin:   admin@albatros.sa   /   Admin@123
     Seller:  seller@albatros.sa  /   Seller@123
     Buyer:   buyer@albatros.sa   /   Buyer@123

   الباسوردز كلها: Admin@123 - Seller@123 - Buyer@123


------------------------------------------------
[6] لو عايز تجرب التسجيل العادي (OTP)
------------------------------------------------

  - اضغط Sign Up
  - املأ البيانات
  - هتوصل لك رسالة على إيميلك فيها رمز التحقق (OTP)
  - الرمز بيتبعت من ahmed1712ax@gmail.com


------------------------------------------------
[7] لو عايز تجرب Google Login
------------------------------------------------

  - اضغط Login ثم "Sign in with Google"
  - هتتحول لصفحة Google عشان تختار حسابك
  - لازم Google Cloud Console يبقى مضيف
    http://localhost:5279/signin-google
    في Authorized redirect URIs


------------------------------------------------
ملاحظات عامة
------------------------------------------------

  - قاعدة البيانات بتتعمل أتوماتيك (مافيش داعي تعمل Create Database)
  - الجداول والبيانات الأولى بتتعمل مع أول تشغيل
  - لو حصل خطأ من أول مرة، وقف التشغيل (Ctrl+C) وشغّله تاني
  - لو غيرت connection string، وقف وشغّل المشروع تاني
